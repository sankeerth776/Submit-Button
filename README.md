# ✨ Animated Submit Button with Glowing Bars

This project is a simple, beautiful, and animated **"Submit" button** built using pure **HTML**, **CSS**, and a bit of **JavaScript**. It features interactive glowing vertical bars that animate on hover, giving a futuristic visual effect.

Perfect for adding to modern websites, portfolios, or UI component libraries.

---

## 🌟 Features

- 💡 Custom-built animated `Submit` button
- 🌈 Glowing vertical bar animation using pure CSS
- 🎨 Gradient and glow hover effects
- ⚙️ Dynamic span generation with JavaScript
- ✨ Responsive, clean layout
- 🎯 No external libraries or frameworks required

---

## 🔧 Technologies Used

- ✅ HTML5
- ✅ CSS3 (with transitions and animations)
- ✅ JavaScript (vanilla)
- ✅ Google Fonts: **Poppins**

## 🚀 How to Run This Project

### ✅ Option 1: Open on Your Computer

1. Download or clone the repository.
2. Make sure `index.html` and `style.css` are in the same folder.
3. Double-click `index.html` → it will open in your browser.
4. Hover over the button to see the glowing animation! 🌟

---

### ✅ Option 2: Run with VS Code + Live Server

1. Open the folder in **Visual Studio Code**.
2. Install the **Live Server** extension (from Extensions panel).
3. Right-click on `index.html` → choose **"Open with Live Server"**.
4. The page will open in your browser automatically.

---

## 🧠 How It Works

- The `index.html` file contains a single `a.btn` button.
- When the page loads, JavaScript dynamically creates 120 vertical `<span>` elements and positions them left-to-right.
- Each span is animated on hover using CSS to create a glowing line effect that expands vertically.
- Alternating spans grow from bottom or top for added effect.
- The background is a dark shade to highlight the glowing effect.
